# SentinelAI Utility Scripts

This directory contains utility scripts for managing and maintaining the SentinelAI platform.

## Scripts

| Script | Description |
|--------|-------------|
| `seed_db.py` | Seeds the database with sample data for development |
| `generate_report.py` | CLI tool for generating security reports |
| `backup_db.py` | Database backup utility |

> **Note:** Scripts will be added as features are implemented.
